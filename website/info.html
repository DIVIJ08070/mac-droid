Up and running in two minutes

One-time setup, then it just works.

Install on the Mac

Download, unzip, move MacDroid.app to Applications and open it. First launch: right-click → Open to pass Gatekeeper.

Install on the phone

Download the APK and tap it. Allow "install unknown apps" for your browser when Android asks — that's normal for apps outside the Play Store.

Pair

Same Wi-Fi, open both apps. Tap your Mac on the phone, check the 6-digit code matches, click Accept on the Mac. Done — forever.

Grant as you go

Each feature asks for its permission the first time you use it (touchpad → Accessibility, audio → Screen & System Audio Recording, tab sync → Automation / Accessibility).

Questions

The honest fine print.

macOS says it "could not verify MacDroid is free of malware"

MacDroid isn't notarized with Apple (that requires a paid developer account), so Gatekeeper warns on first launch. On macOS 15 (Sequoia) and newer: click Done in the dialog, then open System Settings → Privacy & Security, scroll to the bottom, and click "Open Anyway" next to the MacDroid message. On macOS 13–14 you can instead right-click the app → Open → Open. Either way, it's a one-time step.

To skip the warning entirely, install from Terminal instead (the copy-paste command is in the setup guide) — files fetched with curl aren't quarantined, so Gatekeeper never prompts. Already downloaded the zip? Clear the flag with: xattr -cr /Applications/MacDroid.app

Google Play Protect says "App blocked to protect your device"

MacDroid legitimately uses the microphone, screen capture and an accessibility service (for mic streaming, screen sharing and tab sync) — and since it's sideloaded rather than from the Play Store, Google's scanner flags that combination. To install: tap More details → Install anyway if offered, or temporarily disable scanning in Play Store → profile → Play Protect → settings, install the APK, then re-enable it. Everything the app does stays on your own Wi-Fi — there are no servers and no accounts.

Which versions are supported?

macOS 13 Ventura or newer (Intel and Apple Silicon), Android 10 or newer.

Does it work away from home?

Both devices need to be on the same Wi-Fi network. Internet/relay support is on the roadmap.

Is my data safe?

Nothing leaves your local network and there are no servers involved. Traffic between the devices is not yet encrypted, so use it on networks you trust (your home Wi-Fi) — TLS is on the roadmap.

How do I use the phone as a real microphone in Zoom?

Install the free virtual audio driver BlackHole, select it as the mic output device in MacDroid, then pick "BlackHole" as the microphone in Zoom.

Why does tab sync need an accessibility service?

Android offers no official way for one app to read the browser's current address, so MacDroid uses an accessibility service that reads only the address bar of Chrome / Samsung Internet. Nothing else is read or stored.
